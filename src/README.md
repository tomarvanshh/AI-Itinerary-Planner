# AI Itinerary Planner

A small Flask app to generate travel itineraries (local project).

## Quick start

1. Create and activate a virtual environment:
   - Windows (PowerShell): `python -m venv venv; venv\Scripts\Activate.ps1`
2. Install dependencies (create a `requirements.txt` if you have one):
   - `pip install -r requirements.txt` or at minimum `pip install flask`
3. Run the app:
   - `flask run`
